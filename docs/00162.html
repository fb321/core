nvm_sst26_read_write

This application uses multi instances of the Memory driver to communicate with the NVM and the SST26 Flash memories in synchronous mode of operation in RTOS Environment. 

It performs block Erase/Write/Read operations on both the media's. 

 

The application consists of three tasks which are called in the RTOS thread context:

  1. APP_SST26_Tasks() : Performs operations on the SST26 QSPI/SQi Flash Memory.
  2. APP_NVM_Tasks() : Performs operations on the NVM.
  3. APP_MONITOR_Tasks(): Monitors the state of above two Tasks.
