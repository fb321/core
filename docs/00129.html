System Configuration

Configurations for driver instances, polled/interrupt mode, etc.

Macros
 
Name 
Description 
 
DRV_I2S_INDEX_x 
I2S Index selection 
 
DRV_I2S_CLIENTS_NUMBER_IDXx 
Sets up the maximum number of clients that can be connected to the driver 
 
DRV_I2S_QUEUE_DEPTH_COMBINED 
Number of entries of all queues in all instances of the driver. 
 
DRV_I2S_QUEUE_SIZE_IDXx 
Number of entries of all queues in all instances of the driver. 
 
DRV_I2S_DATA_LENGTH_IDXx 
Macro to define the data length for the I2S peripheral 
 
DRV_I2S_INT_SRC_IDXx 
Macro to define the interrupt source 
 
DRV_I2S_XMIT_DMA_CH_IDXx 
Macro to defines the I2S Driver Transmit DMA Channel 
 
DRV_I2S_RCV_DMA_CH_IDXx 
Macro to defines the I2S Driver Receive DMA Channel
