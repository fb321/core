SYS_PORT_PinClear Function

This function drives '0' on the selected I/O line/pin.

C
static inline void SYS_PORT_PinClear(
    SYS_PORT_PIN pin
);
Preconditions

None.

Parameters
Parameters 
Description 
pin 
One of the IO pins from the enum SYS_PORT_PIN 
Returns

None.

Remarks

None.

Example
SYS_PORT_PinClear(SYS_PORT_PIN_PB3);
