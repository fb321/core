MPLAB Harmony Configuration
Components Used:
  • Time System Service
  • STDIO

 

The following components used may vary based on the project configuration selected.

  • TC peripheral library Or Core timer
  • USART, SERCOM or UART peripheral library
  • Board Support Package (BSP)
  • FreeRTOS

 

Other MHC Settings:
  • Configure pins for the USART, SERCOM, UART or DBGU peripheral in Pin Settings

 

NOTE: For projects that use IAR Embedded work bench as IDE, the harmony project file should be opened using MHC plugin (standalone mode) to access the project graph.
