Description 
nvm_sst26_read_write 
This example application shows how to use the Memory driver in asynchronous mode to perform block operations on the NVM and the SST26 media's.

This example application shows how to use the Memory driver in asynchronous mode to perform block operations on the MX25L QSPI flash and the AT25DF SPI flash media.
