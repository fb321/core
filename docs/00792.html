Running The Application
  1. Connect a micro/mini USB cable to the DEBUG port.
  2. Build and Program the application using the MPLAB X IDE.
  3. Press Switch to start the file operations
  4. The LED is turned ON when the File "FILE.TXT" has the app data "Hello World" written in it
